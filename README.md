# 🛒 Lista de Compras (Python)
Criei esse script para praticar a manipulação de listas e tratamento de erros no Python. A ideia é ser um gerenciador rápido via terminal onde eu consiga adicionar tudo de uma vez, remover o que já comprei e ver a lista organizada.

## 🛠️ O que usei aqui:
* **Tratamento de Erros:** Usei `try/except` para o código não quebrar se eu tentar deletar algo que não existe na lista.
* **Loops & Condicionais:** Estruturei com `while True` para o app rodar até eu decidir sair.
* **Manipulação de Strings:** Usei `.split(',')` para permitir adicionar vários itens de uma só vez (ex: "pão, leite, café") e o `.strip()` para limpar espaços inúteis.

## 🚀 Como rodar
1. Tenha o Python instalado.
2. Baixe o arquivo `lista.py`.
3. Rode o comando:
   ```bash
   python lista.py
